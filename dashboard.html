Project 01 World Bank Global Indicators
Primary Tools Python • Plotly • Data Cleaning
Status ● Live Interactive
Analysis Objectives

This interactive dashboard provides a deep dive into global economic and social indicators sourced from the World Bank Open Data. The analysis focuses on understanding trends in GDP growth, population dynamics, and sustainable development metrics across different regions.

Key Technical Highlights
  • Data Cleaning: Handled missing values and standardized regional naming using Python Pandas.
  • Visualization: Implemented Plotly.js for high-fidelity interactive charting.
  • Responsiveness: Dashboard layout scales seamlessly for desktop and mobile viewing.
